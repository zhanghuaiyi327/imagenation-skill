# 性格特征提取提示词

## 目标
基于大五人格模型和关系特定特征，系统提取和分析性格特征，为生成理想伴侣档案提供基础。

## 使用场景
在完成前任分析后，用于提取性格特征并构建性格档案。

## 分析框架

### 1. 大五人格模型分析

#### 1.1 外向性 (Extraversion)

**分析维度：**
- 社交活跃度
- 能量来源（独处 vs 社交）
- 表达倾向
- 冒险精神

**评估标准：**
- **高外向性**：喜欢社交、热情、健谈、积极
- **低外向性**：喜欢独处、安静、保守、内向

**分析提示：**
```
评估外向性水平：

1. 社交行为分析：
   - 社交频率和偏好
   - 社交场合表现
   - 社交能量消耗

2. 表达方式分析：
   - 言语表达频率
   - 情感表达程度
   - 观点表达主动性

3. 活动偏好分析：
   - 喜欢集体活动还是个人活动
   - 对新体验的开放程度
   - 冒险倾向

4. 综合评分：1-10分
   - 1-3分：明显内向
   - 4-6分：适中
   - 7-10分：明显外向
```

#### 1.2 宜人性 (Agreeableness)

**分析维度：**
- 合作倾向
- 同理心水平
- 信任程度
- 利他行为

**评估标准：**
- **高宜人性**：合作、友善、同情、信任
- **低宜人性**：竞争、怀疑、冷漠、自我中心

**分析提示：**
```
评估宜人性水平：

1. 人际关系分析：
   - 合作意愿和表现
   - 冲突处理方式
   - 关系维护努力

2. 情感反应分析：
   - 同理心表现
   - 情感支持提供
   - 他人需求关注

3. 价值观分析：
   - 信任他人程度
   - 利他行为频率
   - 公平正义感

4. 综合评分：1-10分
   - 1-3分：明显低宜人性
   - 4-6分：适中
   - 7-10分：明显高宜人性
```

#### 1.3 尽责性 (Conscientiousness)

**分析维度：**
- 组织能力
- 责任感
- 目标导向
- 自律程度

**评估标准：**
- **高尽责性**：有条理、可靠、自律、目标明确
- **低尽责性**：随意、不可靠、拖延、无计划

**分析提示：**
```
评估尽责性水平：

1. 工作习惯分析：
   - 任务完成可靠性
   - 时间管理能力
   - 工作质量标准

2. 生活管理分析：
   - 日常生活组织
   - 财务规划能力
   - 环境整洁度

3. 目标追求分析：
   - 目标设定和追求
   - 自律和坚持
   - 承诺履行

4. 综合评分：1-10分
   - 1-3分：明显低尽责性
   - 4-6分：适中
   - 7-10分：明显高尽责性
```

#### 1.4 情绪稳定性 (Neuroticism)

**分析维度：**
- 情绪波动
- 压力反应
- 焦虑程度
- 情绪恢复

**评估标准：**
- **高情绪稳定性**：平静、稳定、抗压、乐观
- **低情绪稳定性**：焦虑、情绪化、敏感、易压力

**分析提示：**
```
评估情绪稳定性：

1. 日常情绪分析：
   - 情绪波动频率
   - 情绪强度控制
   - 情绪表达方式

2. 压力反应分析：
   - 压力下的行为变化
   - 压力应对策略
   - 压力恢复速度

3. 心理状态分析：
   - 焦虑和担忧程度
   - 自信心水平
   - 乐观悲观倾向

4. 综合评分：1-10分
   - 1-3分：明显低稳定性（高神经质）
   - 4-6分：适中
   - 7-10分：明显高稳定性（低神经质）
```

#### 1.5 开放性 (Openness)

**分析维度：**
- 好奇心
- 创造力
- 审美敏感度
- 思想开放度

**评估标准：**
- **高开放性**：好奇、创意、艺术、思想开放
- **低开放性**：传统、实际、保守、不喜欢变化

**分析提示：**
```
评估开放性水平：

1. 认知风格分析：
   - 对新事物的好奇心
   - 抽象思维能力
   - 复杂问题处理

2. 审美和创意分析：
   - 艺术欣赏能力
   - 创意表达倾向
   - 想象力丰富度

3. 价值观和信念分析：
   - 对新观念的接受度
   - 传统 vs 创新倾向
   - 多样性接受程度

4. 综合评分：1-10分
   - 1-3分：明显低开放性
   - 4-6分：适中
   - 7-10分：明显高开放性
```

### 2. 关系特定特征分析

#### 2.1 情感表达风格

**分析维度：**
- 情感表达频率
- 表达方式（言语/行动）
- 情感深度
- 表达真实性

**分析提示：**
```
分析情感表达风格：

1. 表达方式分析：
   - 主要表达渠道
   - 表达直接程度
   - 表达一致性

2. 情感内容分析：
   - 表达的情感类型
   - 情感深度和真实性
   - 情感需求表达

3. 关系影响分析：
   - 对关系亲密度的贡献
   - 可能的情感表达问题
   - 改进建议
```

#### 2.2 亲密需求模式

**分析维度：**
- 亲密需求强度
- 亲密表达方式
- 个人空间需求
- 依赖程度

**分析提示：**
```
分析亲密需求模式：

1. 需求强度分析：
   - 对亲密关系的需求程度
   - 独处需求程度
   - 平衡点寻找

2. 表达方式分析：
   - 亲密行为表现
   - 情感连接建立
   - 关系边界设定

3. 关系动态分析：
   - 对关系稳定性的影响
   - 可能的亲密关系问题
   - 健康亲密关系建议
```

#### 2.3 边界意识

**分析维度：**
- 个人边界清晰度
- 他人边界尊重
- 边界沟通能力
- 边界灵活性

**分析提示：**
```
分析边界意识：

1. 自我边界分析：
   - 个人空间需求
   - 隐私保护程度
   - 自我表达边界

2. 他人边界分析：
   - 对他人边界的敏感度
   - 边界尊重程度
   - 边界冲突处理

3. 关系边界分析：
   - 关系角色清晰度
   - 责任和义务边界
   - 关系发展边界
```

#### 2.4 支持行为模式

**分析维度：**
- 支持提供频率
- 支持方式（情感/实际）
- 支持有效性
- 支持接受能力

**分析提示：**
```
分析支持行为模式：

1. 支持提供分析：
   - 支持行为频率
   - 支持方式多样性
   - 支持时机把握

2. 支持质量分析：
   - 支持的实际效果
   - 支持的针对性
   - 支持的持续性

3. 支持接受分析：
   - 接受支持的能力
   - 支持需求表达
   - 支持平衡维护
```

### 3. 性格特征整合

#### 3.1 特征提取规则

**数据来源优先级：**
1. 用户明确描述的特征
2. 具体行为例子中推断的特征
3. 自动化数据分析发现的模式
4. 基于大五人格模型的推断

**特征权重分配：**
- 重复出现的特征：高权重
- 有具体例子支持的特征：高权重
- 用户强调的特征：高权重
- 推断的特征：中低权重

#### 3.2 特征分类系统

**核心性格特征：**
- 大五人格维度得分
- 基本性格倾向
- 核心行为模式

**关系特定特征：**
- 情感表达风格
- 亲密需求模式
- 边界意识
- 支持行为模式

**情境特征：**
- 特定情境下的行为表现
- 压力反应模式
- 社交场合表现

- [ ] 分析逻辑清晰

### 2. 报告完整性检查
- [ ] 所有分析维度都覆盖
- [ ] 特征描述具体明确
- [ ] 建议切实可行

### 3. 实用性检查
- [ ] 报告对用户有实际价值

## 输出格式规范

### 1. 结构要求
- 使用标准模板结构
- 各部分长度均衡
- 标题层级清晰

### 2. 语言要求
- 专业但易懂
- 具体不模糊
- 建设性不批判

### 3. 长度要求
- 总体报告：1000-1500字
- 每个主要部分：200-300字
- 每个子部分：50-100字
