# 兼容性分析提示词

## 目标
基于用户的需求、前任分析和性格特征，全面评估理想伴侣与用户的兼容性，识别匹配点和潜在冲突。

## 使用场景
在完成性格特征分析后，用于评估兼容性和关系发展潜力。

## 分析框架

### 1. 价值观兼容性分析

#### 1.1 核心价值观匹配

**分析维度：**
- 家庭观念一致性
- 事业追求兼容性
- 个人成长理念
- 社会责任认知

**数据来源：**
- 用户的生活方式偏好
- 前任的价值观分析
- 用户的具体要求
- 经验教训中的价值观线索

**分析提示：**
```
评估核心价值观匹配度：

1. 价值观提取：
   - 从用户需求中提取核心价值观
   - 从前任分析中识别价值观模式
   - 从经验教训中总结价值观教训

2. 匹配度分析：
   - 价值观一致性程度
   - 价值观冲突点识别
   - 价值观互补可能性

3. 兼容性评分：1-10分
   - 1-3分：明显不匹配
   - 4-6分：部分匹配，需要调整
   - 7-10分：高度匹配
```

#### 1.2 关系价值观评估

**分析维度：**
- 忠诚和信任观念
- 沟通和表达理念
- 个人空间和亲密平衡
- 冲突解决哲学

**分析提示：**
```
分析关系价值观兼容性：

1. 关系理念提取：
   - 用户对理想关系的描述
   - 前任关系中的价值观体现
   - 用户的具体关系要求

2. 理念匹配分析：
   - 关系期望一致性
   - 关系行为兼容性
   - 关系发展理念契合度

3. 关系价值观评分：1-10分
   - 评分依据：[具体分析]
   - 匹配优势：[列出]
   - 潜在问题：[识别]
```

### 5. 潜在冲突识别

#### 5.1 明确冲突点

**分析维度：**
- 价值观直接冲突
- 性格明显不匹配
- 生活方式根本差异
- 未来发展目标矛盾

**分析提示：**
```
识别明确冲突点：

1. 冲突检测：
   - 对比用户需求和理想伴侣特征
   - 识别直接矛盾点
   - 评估冲突严重程度

2. 冲突分析：
   - 冲突的根本原因
   - 冲突的影响范围
   - 冲突解决难度

3. 冲突严重性评级：
   - 轻微冲突：可以轻松调整
   - 中等冲突：需要努力解决
   - 严重冲突：可能影响关系存续
```

#### 5.2 潜在风险点

**分析维度：**
- 隐性价值观差异
- 性格潜在冲突
- 生活习惯细微不协调
- 未来可能的发展分歧

**分析提示：**
```
识别潜在风险点：

1. 风险检测：
   - 分析细微差异
   - 预测未来可能问题
   - 识别敏感领域

2. 风险评估：
   - 风险发生概率
   - 风险影响程度
   - 风险预防难度

3. 风险等级：
   - 低风险：容易管理
   - 中风险：需要关注
   - 高风险：需要主动预防
```

## 输出格式规范

### 1. 结构要求
- 使用标准报告模板
- 各部分逻辑连贯
- 标题层级清晰

### 2. 语言要求
- 客观准确
- 具体明确
- 建设性导向

### 3. 长度要求
- 总体报告：1200-1800字
- 每个主要部分：250-350字
- 每个子部分：80-120字
